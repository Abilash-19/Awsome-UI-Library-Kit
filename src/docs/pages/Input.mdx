## Basic Usage

A labeled input with placeholder and helper text.

<ComponentPreview code={`<Input
  label="Email"
  placeholder="you@company.com"
  helperText="We won't share this."
/>`}>
  <div className="w-full max-w-sm mx-auto">
    <Input label="Email" placeholder="you@company.com" helperText="We won't share this." />
  </div>
</ComponentPreview>

## Icons & Loading

Add context with leading/trailing icons, or show a spinner during validation.

<ComponentPreview code={`<Input label="Search" leftIcon={<span>🔍</span>} placeholder="Search..." />
<Input label="Password" rightIcon={<span>👁️</span>} type="password" />
<Input isLoading label="Validating..." />`}>
  <div className="w-full max-w-sm mx-auto space-y-4">
    <Input label="Search" leftIcon={<span>🔍</span>} placeholder="Search..." />
    <Input label="Password" rightIcon={<span className="opacity-40">👁️</span>} type="password" placeholder="••••••••" />
    <Input isLoading label="Checking" placeholder="Validating..." defaultValue="abilash_19" />
  </div>
</ComponentPreview>

## Validation

Error messages replace helper text and recolor the border.

<ComponentPreview code={`<Input label="Username" error="Already taken" defaultValue="admin" />`}>
  <div className="w-full max-w-sm mx-auto">
    <Input label="Username" error="This username is unavailable." defaultValue="admin" />
  </div>
</ComponentPreview>
