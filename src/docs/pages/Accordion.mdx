Accordions are vertically collapsing containers that manage large amounts of content in a compact footprint. They are perfect for FAQ sections, settings panels, and detail-rich lists.

## Basic Usage
A standard accordion includes a title, an optional subtitle, and the body content.

<ComponentPreview code={`<Accordion 
  title="What is this UI Library?"
  subtitle="Commonly asked questions"
>
  This library is a collection of high-performance, theme-aware React components 
  styled with Tailwind CSS and advanced CSS animations.
</Accordion>`}>
  <div className="w-full max-w-2xl mx-auto">
    <Accordion 
      title={<Typography variant="h6" weight="bold">What is this UI Library?</Typography>}
      subtitle="Frequently Asked Questions"
    >
      <div className="py-2 opacity-80 leading-relaxed">
        Our library is built on top of modern standards using React and Tailwind. 
        Every component is designed to be accessible, performant, and deeply customizable 
        via our integrated theming engine.
      </div>
    </Accordion>
  </div>
</ComponentPreview>

## Customization
### Borderless Style
For a more integrated look, use the `borderless` variant.

<ComponentPreview code={`<Accordion variant="borderless" title="Advanced Settings">
  Content here...
</Accordion>`}>
  <div className="w-full max-w-2xl mx-auto space-y-0">
    <Accordion variant="borderless" title="Display Settings">
      <div className="py-2">Modify your theme and layout preferences here.</div>
    </Accordion>
    <Accordion variant="borderless" title="Privacy Controls">
      <div className="py-2">Manage your data and notification settings.</div>
    </Accordion>
  </div>
</ComponentPreview>

### Default Open & Disabled
Control the initial state or prevent user interaction.

<ComponentPreview code={`<Accordion isDefaultOpen title="Open by Default">...</Accordion>
<Accordion isDisabled title="Locked Section">...</Accordion>`}>
  <div className="w-full max-w-2xl mx-auto space-y-4">
    <Accordion isDefaultOpen title="Expansion Test">
      This accordion starts in the open state.
    </Accordion>
    <Accordion isDisabled title="Locked Section">
      You cannot open this section until criteria are met.
    </Accordion>
  </div>
</ComponentPreview>
