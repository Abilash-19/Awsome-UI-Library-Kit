## Basic Usage

Click the header to expand or collapse the content.

<ComponentPreview code={`<Accordion title="What is this library?">
  A collection of theme-aware React components.
</Accordion>`}>
  <div className="w-full max-w-2xl mx-auto">
    <Accordion
      title={<Typography variant="h6" weight="bold">What is this library?</Typography>}
      subtitle="FAQ"
    >
      <div className="py-2 opacity-80 leading-relaxed text-sm">
        A collection of accessible, theme-aware React components built with Tailwind CSS.
      </div>
    </Accordion>
  </div>
</ComponentPreview>

## Borderless

Use `variant="borderless"` for a cleaner inline look.

<ComponentPreview code={`<Accordion variant="borderless" title="Settings">
  Content...
</Accordion>`}>
  <div className="w-full max-w-2xl mx-auto">
    <Accordion variant="borderless" title="Display Settings">
      <div className="py-1 text-sm">Adjust theme and layout preferences.</div>
    </Accordion>
    <Accordion variant="borderless" title="Privacy Controls">
      <div className="py-1 text-sm">Manage data and notification settings.</div>
    </Accordion>
  </div>
</ComponentPreview>
