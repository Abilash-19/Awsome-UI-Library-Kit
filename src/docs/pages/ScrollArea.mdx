ScrollArea targets large content blocks by providing a smooth scroll container with reveal-on-scroll animations. It helps keep long lists or documents contained while maintaining a premium feel.

## Basic Usage
Wrap your content in a `ScrollArea` and use `ScrollAreaItem` (or `ScrollArea.Item`) for individual items you want to animate.

<ComponentPreview code={`import { ScrollArea, ScrollElement } from 'ui-library';

<ScrollArea className="h-48 border rounded-xl p-4">
  <div className="space-y-4">
    <ScrollElement>
      <div className="h-20 bg-primary-500/10 rounded-lg flex items-center justify-center">Item 1</div>
    </ScrollElement>
    <ScrollElement>
      <div className="h-20 bg-primary-500/10 rounded-lg flex items-center justify-center">Item 2</div>
    </ScrollElement>
  </div>
</ScrollArea>`}>
  <ScrollArea className="h-48 border rounded-xl p-4 w-full">
    <div className="space-y-8">
      <ScrollAreaItem>
        <div className="h-24 bg-pink-500/5 border border-pink-500/10 rounded-2xl flex items-center justify-center font-bold">Scroll slowly to see the reveal animation</div>
      </ScrollAreaItem>
      <ScrollAreaItem revealType="up">
        <div className="h-24 bg-pink-500/5 border border-pink-500/10 rounded-2xl flex items-center justify-center font-bold">Item 2 (Reveal Up)</div>
      </ScrollAreaItem>
      <ScrollAreaItem revealType="fade">
        <div className="h-24 bg-pink-500/5 border border-pink-500/10 rounded-2xl flex items-center justify-center font-bold">Item 3 (Fade In)</div>
      </ScrollAreaItem>
      <ScrollAreaItem revealType="blur">
        <div className="h-24 bg-pink-500/5 border border-pink-500/10 rounded-2xl flex items-center justify-center font-bold">Item 4 (Blur Entrance)</div>
      </ScrollAreaItem>
    </div>
  </ScrollArea>
</ComponentPreview>

## Customization
### Animation Presets
You can specify the entrance effect per item.

| Type | Description |
| :--- | :--- |
| `up` / `down` | Horizontal slide with fade |
| `left` / `right` | Vertical slide with fade |
| `fade` | Simple opacity transition |
| `scale` | Zoom-in effect |
| `blur` | Soft blur-to-clear transition |
