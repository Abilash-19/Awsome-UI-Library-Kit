## Infinite Scroll

Use the `onScroll` callback to detect when the user reaches the bottom and load more items.

<ComponentPreview code={`function InfiniteList() {
  const [items, setItems] = useState(initialItems);

  const handleScroll = (e) => {
    const el = e.currentTarget;
    if (el.scrollTop + el.clientHeight >= el.scrollHeight - 40) {
      loadMore(); // fetch and append items
    }
  };

  return (
    <ScrollArea className="h-72" onScroll={handleScroll}>
      {items.map(item => (
        <ScrollElement key={item.id} revealType={item.type}>
          <div>{item.title}</div>
        </ScrollElement>
      ))}
    </ScrollArea>
  );
}`}>
  <InfiniteScrollDemo />
</ComponentPreview>

## Animation Types

| Type | Effect |
| :--- | :--- |
| `up` / `down` | Slide with fade |
| `left` / `right` | Horizontal slide |
| `fade` | Opacity only |
| `scale` | Zoom in |
| `blur` | Blur to clear |
