## Basic Usage

<ComponentPreview code={`<RadioGroup
  label="Fruit"
  options={[
    { id: '1', value: 'apple', label: 'Apple', description: 'Sweet' },
    { id: '2', value: 'banana', label: 'Banana', description: 'Soft' },
  ]}
  defaultValue="apple"
/>`}>
  <RadioGroup
    label="Favorite Fruit"
    options={[
      { id: '1', value: 'apple', label: 'Apple', description: 'Sweet and crunchy' },
      { id: '2', value: 'banana', label: 'Banana', description: 'Yellow and soft' },
    ]}
    defaultValue="apple"
  />
</ComponentPreview>

## Card Variant

A more prominent, button-like selection style.

<ComponentPreview code={`<RadioGroup variant="card" options={[...]} />`}>
  <RadioGroup
    variant="card"
    options={[
      { id: '1', value: 'sm', label: 'Small', description: 'Up to 5 users' },
      { id: '2', value: 'md', label: 'Medium', description: 'Up to 20 users' },
    ]}
    defaultValue="md"
  />
</ComponentPreview>
