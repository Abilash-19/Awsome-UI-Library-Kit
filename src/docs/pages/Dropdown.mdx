## Basic Usage

Use `selectedValue` and `onChange` to control the selected option. Options support `label`, `value`, and `disabled`.

<ComponentPreview code={`const [value, setValue] = useState("");


<Dropdown
  label="Framework"
  required
  placeholder="Pick a framework"
  options={[
    { label: "React", value: "react" },
    { label: "Vue", value: "vue" },
    { label: "Svelte", value: "svelte" },
    { label: "Angular", value: "angular" },
  ]}
  selectedValue={value}
  onChange={setValue}
/>`}>
  <BasicDemo />
</ComponentPreview>

## With Icons

Each option can include an `icon` prop. The icon is displayed in both the trigger and the dropdown list.

<ComponentPreview code={`<Dropdown
  label="Payment Method"
  placeholder="Select payment"
  options={[
    { label: "Credit Card", value: "card", icon: <CreditCardIcon /> },
    { label: "Bank Transfer", value: "bank", icon: <BankIcon /> },
    { label: "Crypto", value: "crypto", icon: <CryptoIcon />, disabled: true },
  ]}
  selectedValue={value}
  onChange={setValue}
/>`}>
  <IconDemo />
</ComponentPreview>

## Searchable

Add `searchable` to enable a filter input inside the panel. Options are filtered as you type, with keyboard navigation still fully supported.

<ComponentPreview code={`<Dropdown
  label="Country"
  placeholder="Search & select country"
  searchable
  searchPlaceholder="Type to filter..."
  options={countries.map(c => ({ label: c, value: c.toLowerCase() }))}
  selectedValue={value}
  onChange={setValue}
/>`}>
  <SearchDemo />
</ComponentPreview>

## Scrollable List

When options exceed the panel height, a themed scrollbar appears. Keyboard navigation (Arrow keys, Home, End) auto-scrolls the highlighted item into view.

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `Arrow Down` | Open dropdown / highlight next option |
| `Arrow Up` | Open dropdown / highlight previous option |
| `Enter` / `Space` | Select highlighted option / open dropdown |
| `Escape` | Close dropdown |
| `Home` | Jump to first enabled option |
| `End` | Jump to last enabled option |
| Type characters | Filter options (when `searchable` is enabled) |
