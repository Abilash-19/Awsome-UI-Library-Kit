The library is built on a robust, token-based theming system that supports light mode, dark mode, and custom brand overrides.

## Theme Provider
To use the theme system, wrap your application in the `ThemeProvider`.

```tsx
import { ThemeProvider } from 'ui-library';

function App() {
  return (
    <ThemeProvider defaultThemeName="light">
      <YourAppContents />
    </ThemeProvider>
  );
}
```

## How it Works
The theme system uses **CSS Variables** defined in `:root` and `.dark` classes (or custom theme classes). This means:
- **No extra JS weight** for runtime style calculations.
- **Zero Flash of Unstyled Content** (FOUC).
- **Infinite customization** — just override the CSS variables.

## Dynamic Overrides
You can live-update the theme programmatically using the `useTheme` hook. This is what powers the theme picker in the Playground.

```tsx
const { overrideTheme, resetTheme } = useTheme();

const oceanTheme = {
  palette: {
    primary: { 500: '#0077ff' }
  }
};

<Button onClick={() => overrideTheme(oceanTheme)}>Go Blue</Button>
```
