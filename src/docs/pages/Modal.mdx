## Basic Usage

Control visibility with `isOpen` and `onClose`.

<ComponentPreview code={`const [isOpen, setIsOpen] = useState(false);

<Button onClick={() => setIsOpen(true)}>Open</Button>
<Modal isOpen={isOpen} onClose={() => setIsOpen(false)} title="Confirm">
  Are you sure?
</Modal>`}>
  <ModalDemo title="Example Modal" />
</ComponentPreview>

## Sizes

Four widths: `sm`, `md`, `lg`, and `xl`.

<ComponentPreview code={`<Modal size="sm" ... />
<Modal size="lg" ... />`}>
  <div className="flex flex-wrap gap-3">
    <ModalDemo size="sm" title="Small" />
    <ModalDemo size="md" title="Medium" />
    <ModalDemo size="lg" title="Large" />
  </div>
</ComponentPreview>

## Without Close Button

Set `showCloseButton={false}` to require an explicit action.

<ComponentPreview code={`<Modal showCloseButton={false} title="Required Action" />`}>
  <ModalDemo showCloseButton={false} title="Required Action" />
</ComponentPreview>
